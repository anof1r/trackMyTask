### 7. Floating Action Button

#### ДО:
```
  [+]  <- Простая синяя кнопка
```

#### ПОСЛЕ:
```
  [+]  <- Градиент, вращение при hover
       <- Увеличение при hover
       <- Тень с цветом primary
```

Анимация:
```
Обычное состояние:    scale(1.0)
Hover:                scale(1.1) rotate(90deg)
Active:               scale(0.95) rotate(90deg)
```

## 🎬 Анимации

### Типы анимаций:

1. **Hover эффекты**
   - Cards: `translateY(-2px)` + shadow
   - Menu items: `translateX(4px)` + background
   - Buttons: `scale(1.1)` + shadow

2. **Drag & Drop**
   - Preview: `rotate(3deg)` + opacity(0.9)
   - Placeholder: dashed border + opacity(0.5)
   - Transitions: `cubic-bezier(0.4, 0, 0.2, 1)`

3. **Loading**
   - Spinner: `rotate(360deg)` infinite
   - Fade in: opacity 0 → 1

4. **Errors**
   - Slide down: `translateY(-20px)` → `translateY(0)`
